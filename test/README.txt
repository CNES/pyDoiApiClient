prerequisites:

The server DOI should be launched with the test configuration 
ex: "java -jar server_DOI.jar -f config-test.properties", hence it will be connected 
the testing database.

The user "test" and the project "CFOSAT, 828606" should be added to the testing database
The token present in TestMdsApi should be added to the testing datbase

To install the python library you should run this command:
"sudo python setup.py install" from the setup.py directory, the library will be installed as system library, the library location could be customized via --home option
